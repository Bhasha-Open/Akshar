Morphology API#

Overview#

Helpers for basic morphological segmentation and lemmas where available.

Quick Reference#

Parameters

Toy Examples#

from akshar.morph import segment_morphemes, guess_lemma, analyze_morphology
print(segment_morphemes("अधिकारियों", language="hindi"))
print(guess_lemma("अधिकारियों", language="hindi"))
print(analyze_morphology("भारत अधिकारियों"))

Per‑function guides#

segment_morphemes#

  • Break a word into plausible morphemes using lightweight rules.

  • Prefers conservative splits; avoids over‑segmenting short words.

  • Works best on Hindi/Sanskrit nouns/verbs.

  • Feed normalized input for stable output.

guess_lemma#

  • Heuristic lemma finder; falls back to the original word.

  • Avoids lossy guesses; returns None if uncertain.

  • Good for grouping inflected forms in dashboards.

  • Not a full‑blown lemmatizer.

analyze_morphology#

  • Tokenize a string and attach morpheme splits per token.

  • Returns a list of (word, parts) for quick inspection.

  • Handy for notebooks and error analysis.

  • Combine with segmentation and normalization first.

Morphological segmentation using (optional) Morfessor models.

If models are not available, we fall back gracefully and return whole words so the caller doesn’t get surprised by missing dependencies.

Morphology API#

Overview#

Optional Morfessor-backed segmentation for Hindi/Sanskrit. Gracefully degrades to word-level if models are not available.

Examples#

from akshar.morph import segment_hindi, segment_sanskrit, get_hindi_segmenter
segment_hindi("लोकतांत्रिक")

seg = get_hindi_segmenter()
print(seg.is_model_loaded())

Recipes#

  • Use morphology when available else fallback:

    from akshar.segment import word_tokenize
word_tokenize("भारत लोकतांत्रिक देश है", language="hindi", use_morphology=True)

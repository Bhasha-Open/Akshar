API#

Visualization helpers for Akshar outputs.

Small, dependency-light utilities to visualize script segments, akshar boundaries, tokens, and simple stats either in terminal or HTML contexts.

colorize_by_script(text_segments)#

Generate ANSI colored output for terminal display.

Parameters:

text_segments – List of (text, script) tuples from detect_code_switches

Returns:

String with ANSI color codes

export_analysis_json(analysis, output_path)#

Export analysis to JSON file for external tools.

format_akshar_boundaries(akshars)#

Display akshars with boundaries marked.

Example

[क] [्] [ष] [े] [त] [्] [र] [े]

format_token_boundaries(text, tokens)#

Show token boundaries with visual markers.

Example output:

▁aaj | ▁मौसम | ▁बहुत | ▁अच्छा | ▁है

generate_html_visualization(analysis)#

Generate HTML for rich visualization of tokenization.

This can be used in Streamlit or saved as standalone HTML.

print_comparison_table(texts, tokenizer)#

Print comparison table showing how different texts tokenize. Useful for debugging and evaluation.

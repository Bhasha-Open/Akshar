API#

Anusvāra handling for Devanagari.

Resolves anusvāra (ं) to an appropriate homorganic nasal based on the following consonant’s place of articulation. We return both forms so callers can decide how aggressive they want to be when rewriting.

get_nasal_for_consonant(cons)#

Return the homorganic nasal for a given consonant.

Parameters:

cons (str) – Single Devanagari consonant.

Returns:

One of ‘ङ’, ‘ञ’, ‘ण’, ‘न’, ‘म’ or ‘ं’ as fallback.

Return type:

str

resolve_anusvara(text, store_both=True)#

Resolve anusvāra (ं) into a homorganic nasal, conservatively.

Parameters:

  • text (str) – Input string.

  • store_both (bool) – If True, return both ‘original’ and ‘resolved’ (non-destructive).

Returns:

Mapping with ‘original’ and ‘resolved’ keys (or only ‘resolved’).

Return type:

Dict[str, str]

Examples

>>> resolve_anusvara("संस्कृत")
{'original': 'संस्कृत', 'resolved': 'संसकृत'}  # example, depends on following consonant

What you can do in 60 seconds
-----------------------------

.. code-block:: python

   from akshar.tokenizer import aksharTokenizer
   tk = aksharTokenizer()
   tokens = tk.tokenize("आज मौसम अच्छा है")
   print(tokens)

.. code-block:: python

   from akshar.segment import detect_code_switches
   print(detect_code_switches("aaj मौसम accha hai"))
